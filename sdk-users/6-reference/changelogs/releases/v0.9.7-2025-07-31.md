# Kailash SDK v0.9.7 Release Notes
**Release Date:** July 31, 2025  
**Type:** Patch Release (Critical Bug Fix)

## 🚨 Critical Bug Fix

### IterativeLLMAgentNode MCP Fallback Mechanism
**Issue:** IterativeLLMAgentNode returning template responses instead of performing iterative AI reasoning when MCP tools were unavailable
- **Root Cause:** Node was returning hardcoded template "No tools executed for action: X" when MCP tools failed or were unavailable
- **Fix:** Implemented proper LLM fallback mechanism to enable general iterative AI reasoning without requiring MCP servers
- **Impact:** Resolves critical production issue for Treasury migration team and other workflows using IterativeLLMAgentNode
- **Files:** `src/kailash/nodes/ai/iterative_llm_agent.py`

## 🚀 Performance Optimizations

- **Impact:** All 7 cyclic tests now complete in 0.67s total (well under 1s limit)
