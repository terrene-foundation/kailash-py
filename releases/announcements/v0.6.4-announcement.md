# Kailash Python SDK v0.6.4 Released! 🎉

We're excited to announce Kailash Python SDK v0.6.4 with enterprise-grade parameter injection and 100% E2E test coverage!

## Highlights

✨ **Enterprise Parameter Injection** - WorkflowBuilder now supports explicit parameter mapping with dot notation for nested data access
🚀 **100% E2E Test Pass Rate** - All comprehensive end-to-end tests now passing with improved stability
📚 **Documentation Excellence** - Updated based on real E2E test findings with correct patterns and best practices
🔧 **Bug Fixes** - Fixed permission check structures, PythonCodeNode parameter access, and integration test reliability

## Key Improvements

### Simplified Parameter Injection
```python
# Map nested parameters with dot notation
workflow_builder.add_workflow_inputs("processor", {
    "user_id": "data.user_id",
    "config": "settings.processing"
})
```

### Direct PythonCodeNode Access
```python
# Parameters are now in namespace directly
workflow_builder.add_node("PythonCodeNode", "validator", {
    "code": "if not enterprise: raise ValueError('Missing enterprise')"
})
```

### Correct Permission Patterns
```python
# Fixed nested result structure
if result.get("result", {}).get("check", {}).get("allowed", False):
    print("Access granted!")
```
