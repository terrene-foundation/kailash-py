# 🎉 Kailash Python SDK v0.2.0 Released!

We're thrilled to announce the release of **Kailash Python SDK v0.2.0**, featuring our groundbreaking **Universal Hybrid Cyclic Graph Architecture**!

## 🌟 What's New?

### Cyclic Workflows Are Here!
For the first time, Kailash SDK supports iterative, feedback-driven workflows with native cycle support. Build self-improving systems, optimization algorithms, and complex simulations with ease.

### Key Highlights:
- **🔄 CycleAwareNode**: Powerful base class with built-in iteration tracking, state management, and convergence detection
- **⚡ Blazing Fast**: Achieve ~30,000 iterations/second with optimized execution
- **🛠️ Developer Tools**: Comprehensive suite including CycleAnalyzer, CycleDebugger, and CycleProfiler
- **📊 Data Science Ready**: Enhanced PythonCodeNode with full DataFrame and NumPy support
- **✅ Production Tested**: Battle-tested with workflows running 100,000+ iterations

## 💻 Quick Start

```bash
pip install kailash==0.2.0
```

```python
from kailash import Workflow
from kailash.workflow import CycleBuilder

# Build an iterative optimizer in seconds!
builder = CycleBuilder("gradient_descent")
optimizer = builder.add_cycle_aware_node(
    "optimizer",
    code="value = prev_value - 0.1 * gradient"
)
workflow = builder.build(max_iterations=100)
```
