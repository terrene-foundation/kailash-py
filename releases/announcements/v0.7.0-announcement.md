# Kailash Python SDK v0.7.0 Released! 🎉

## Major Framework Release with Infrastructure Excellence

We're thrilled to announce the release of Kailash Python SDK v0.7.0, featuring complete application frameworks, comprehensive infrastructure improvements, and 100% E2E test pass rates!

### 🌟 Highlights

**Complete Application Frameworks**
- **DataFlow**: Zero-config ETL framework with MongoDB-style queries
- **Nexus**: Multi-channel platform supporting API, CLI, and MCP interfaces

**Infrastructure Excellence**
- **100% E2E Test Pass Rate**: Comprehensive infrastructure fixes (up from 20%)
- **Enhanced AsyncNode**: Thread-safe async execution in any context
- **8 New Operations**: Extended monitoring capabilities

**Enterprise Features**
- **Distributed Transactions**: Automatic Saga/2PC pattern selection
- **Query Builder**: MongoDB-style queries across PostgreSQL, MySQL, SQLite
- **Redis Cache**: Enterprise-grade caching with pattern invalidation

### 🚀 Quick Start

```bash
pip install --upgrade kailash==0.7.0
```

#### DataFlow Example
```python
from kailash.dataflow import DataFlow

df = DataFlow()
users = df.users.find({"age": {"$gt": 18}}).cache()
```

#### Nexus Multi-Channel
```python
from kailash.nexus import create_nexus

nexus = create_nexus(
    enable_api=True,
    enable_cli=True,
    enable_mcp=True
)
```
