## Notes

### Release Focus
This release introduces major new infrastructure:
- **Workflow Studio Backend**: Complete REST API for visual workflow building
- **JWT Authentication**: Full authentication system with refresh tokens
- **Access Control**: Role-based and node-level access control
- **Multi-Tenancy**: Complete tenant isolation architecture
- **Documentation**: Comprehensive updates and compliance

### Key Achievements
- Complete JWT authentication system
- Multi-tenant architecture with full isolation
- Role-based access control (RBAC)
- Node-level permissions with output masking
- Access-controlled runtime for transparent security
- Workflow Studio backend API
- All documentation compliant with standards
- All pre-commit checks passing

- Frontend UI components not yet implemented
- AI Assistant not yet integrated
- Datetime comparison bug in list_runs() (to be fixed in v0.1.7)
