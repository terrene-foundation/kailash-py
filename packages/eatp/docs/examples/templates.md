# Constraint Templates

See `examples/constraint_templates.py` for the full runnable example.

EATP includes 6 built-in constraint templates:

| Template | Use Case |
|----------|----------|
| `governance` | Enterprise governance agents |
| `finance` | Financial processing agents |
| `community` | Community platform agents |
| `standards` | Standards compliance agents |
| `audit` | Audit and compliance agents |
| `minimal` | Minimal constraints for development |
