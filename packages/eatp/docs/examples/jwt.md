# JWT Interop

See `examples/jwt_interop.py` for the full runnable example.

Export EATP trust chains as JWT tokens for integration with existing systems:

- JWT: Standard JSON Web Tokens
- W3C VC: Verifiable Credentials
- DID: Decentralized Identifiers
- UCAN: User Controlled Authorization Networks
- SD-JWT: Selective Disclosure JWT
- Biscuit: Biscuit authorization tokens
