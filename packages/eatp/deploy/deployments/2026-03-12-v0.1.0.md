## Features

- 5 core data structures: GenesisRecord, DelegationRecord, CapabilityAttestation, AuditAnchor, ConstraintEnvelope
- 4 trust operations: ESTABLISH, DELEGATE, VERIFY, AUDIT
- Ed25519 cryptographic signing and verification (PyNaCl)
- 3 trust stores: InMemory, Filesystem, PostgreSQL
- 3 enforcement modes: StrictEnforcer, ShadowEnforcer, Decorators
- 6 interop formats: JWT, W3C VC, DID, UCAN, SD-JWT, Biscuit
- Reasoning traces with dual-binding crypto (v2.2 signing payload)
- Confidentiality-based selective disclosure (5 levels)
- Verification gradient: QUICK, STANDARD, FULL
- CLI with 10+ commands
- MCP server (5 tools, 4 resources)
- 1557 tests passing
