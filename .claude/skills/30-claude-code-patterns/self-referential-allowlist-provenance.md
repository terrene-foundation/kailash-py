**Editing contract (READ FIRST):** the LIVE allowlist is the path skeleton in the RULE, not this file. A path written at depth-0 in the rule's bullet is a live firing entry; a path here is documentation only. Any edit to this file fires the Tier-1 self-referential redteam gate (it is under `skills/30-claude-code-patterns/**`, on the Rule-2 Skills allowlist).
