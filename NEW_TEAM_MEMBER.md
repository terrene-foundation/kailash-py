# 🚀 Welcome New Team Member!

## First Step: Identify Your Role

### I'm here to...

**Choose ONE:**
- [ ] **Build solutions using the SDK** → You're an **SDK User**
  - Creating workflows for business problems
  - Using existing nodes and patterns
  - Building on top of Kailash SDK

- [ ] **Develop the SDK itself** → You're an **SDK Contributor**
  - Adding new features to the SDK
  - Fixing bugs in core functionality
  - Improving SDK architecture

- [ ] **Manage team and releases** → You're a **Team Lead/Integration Manager**
  - Planning development work
  - Coordinating team efforts
  - Managing releases and integration

## Second Step: Critical Information

### ⚠️ IMPORTANT: We Use Claude Code for Everything

**What this means:**
- All task management through conversational AI
- No manual editing of TODO files
- No direct GitHub issue creation (except bugs from external users)
- All progress updates through Claude Code

**Why this matters:**
- Ensures consistency across all systems
- Prevents coordination failures
- Enables intelligent workload management
- Maintains complete audit trail
