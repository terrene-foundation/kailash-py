# Deployment Record — kailash-kaizen 2.28.0 (2026-06-19)

Single-package minor release closing issue **#1388** — `LlmClient` lifecycle surface.

## Summary

`kaizen.llm.client.LlmClient` exposed no `close()`/`aclose()`/`__aexit__`
lifecycle method (`dir(LlmClient)` had none). Once a client held a persistent
HTTP transport, the socket was released only at GC — emitting
`ResourceWarning: unclosed <socket.socket ...>` and failing any run under
`-W error::ResourceWarning`. There was also no connection reuse across
`embed()` calls.

2.28.0 closes the API-surface lifecycle gap **additively** (minor bump — new
public API, backward-compatible):

- **`aclose()`** (idempotent) + **`__aenter__`/`__aexit__`** deterministically
  close an opt-in pooled `LlmHttpClient`.
- **Opt-in connection pooling** — `async with LlmClient.from_deployment(d) as c:`
  eagerly creates ONE persistent `LlmHttpClient` on entry (before any concurrent
  `embed()`), reuses it across every `embed()` in the scope (amortizing the
  SSRF-resolver + connection-pool setup, surviving per-call errors), and closes
  it on exit. The pooled transport still routes through `LlmHttpClient`'s
  structurally-installed `SafeDnsResolver` — no SSRF bypass.
- **Backward-compatible** — `embed()`'s transport acquisition is 3-case:
  caller-injected `http_client=` (caller owns), managed (instance-pooled), and
  **unmanaged → UNCHANGED fresh-per-call construct+close**. Existing one-shot
  `await LlmClient.from_deployment(d).embed(...)` callers hold nothing, leak
  nothing, warn nothing.
- **`__del__` is WARN-ONLY**, mirroring `LlmHttpClient.__del__`: a managed client
  that pooled a transport and was never closed emits a `ResourceWarning` naming
  `aclose`; it never calls close from the finalizer (async cleanup in `__del__`
  deadlocks on CPython's root logging lock per `rules/patterns.md` § "Async
  Resource Cleanup"). One-shot callers hold no transport → no warning.
- No sync `close()` (would need `asyncio.run()` and break inside event loops per
  `rules/patterns.md` § "Paired Public Surface"). `__aexit__` params typed with
  `types.TracebackType` (zero net-new type findings vs main).
