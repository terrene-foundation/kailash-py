query. Pre-existing; DIFFERENT bug class from #1117; **plausibly SYSTEMIC** across every rag
node's LLMAgentNode stage (eval judges, privacy, etc. — UNAUDITED). This is the THIRD
"provably correct" layer:

  L2 (#1117): make nodes PUBLISH output.

  output is contextually real, not system-prompt-only.

`messages`? + fix the input contract) — exceeds this shard, likely needs user product input.
Recorded (not deferred-and-forgotten). Value-anchor: brief "provably correct".
