## LOC estimate

- **Cluster A (this file's scope)**: ~40–50 LOC across 6 TYPE FIXes + 7 GUARD FIXes + 1 real bug fix.
- TYPE FIXes are mechanical: single-line annotation changes, zero runtime change.
- GUARD FIXes per Rule 3a need actionable RuntimeError messages — ~4–6 LOC each.

This is Rule 3 silent-fallback territory and is in scope for the typing cleanup but materially changes behavior.

Single shard fits within the ≤500 LOC load-bearing-logic / ≤5–10 invariants budget per `autonomous-execution.md` Rule 1.
