# 🎉 Kailash Python SDK v0.1.0 Release Announcement

## We're Live on PyPI!

The Kailash Python SDK is now officially available on PyPI. This marks a significant milestone in our mission to bridge the gap between rapid prototyping and production-ready workflows.

### 🚀 Get Started Today

```bash
pip install kailash
```

### 📊 Release Statistics

- **Package Size**: 228KB (wheel) / 199KB (source)
- **Dependencies**: Minimal core dependencies with optional extras
- **Python Support**: 3.11+
- **Test Coverage**: 539 tests passing
- **Examples**: 30+ ready-to-use examples

### 🌟 Key Highlights

1. **Production-Ready**: Built from the ground up to align with Kailash container architecture
2. **Developer-Friendly**: Pythonic API that feels natural to data scientists and engineers
3. **Battle-Tested**: Extensively tested with real-world use cases
4. **Well-Documented**: Comprehensive documentation and examples

### 📦 What's Included

- **Core Framework**: Node-based workflow engine
- **Node Library**: 50+ pre-built nodes across 6 categories
- **Runtime Options**: Local, Docker, and parallel execution
- **Monitoring**: Real-time task tracking and performance metrics
- **Export Tools**: Direct export to Kailash-compatible formats
- **CLI**: Command-line interface for workflow operations

### 🎯 Perfect For

- **Rapid Prototyping**: Test ideas quickly without containerization
- **ML Pipelines**: Build complex data processing workflows
- **API Integrations**: Connect to external services with built-in nodes
- **Team Collaboration**: Share workflows between ABCs and PDT
